Oque è?

HTML5, sigla para Hypertext Markup Language, é uma linguagem de marcação de hipertexto para apresentar e estruturar o conteúdo na web. Ou seja, todos os conteudos de um site/aplicativo, sao feitos com o HTML-5.

Como surgiu?

Desde 1999 o HTML ficou estacionado na versão HTML4. As primeiras especificações do HTML5 começaram a surgir em 2008 através do Consórcio de Empresas de Tecnologia W3C. O projeto da nova versão ganhou a mídia a partir de 2010 e desde então vem sendo cada vez mais usado pelos desenvolvedores web
